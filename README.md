# tool-tide 🌊

> *Catch the rising tide — not the famous, but the accelerating.*

**「常時上位」ではなく「今週加速している新星」を捕捉する、技術トレンド自動収集システム。**

---

## これは何か

GitHub Trending が返すのは「今日の有名株」だ。  
このシステムが欲しいのは、**先週まで誰も知らなかった、でも今週スターが加速しているプロジェクト**。

絶対数ではなく「加速度」を測る。若さと内容密度を掛け合わせる。  
その3軸で、埋もれた新星を浮かび上がらせる。

---

## 背景と経緯

このプロジェクトは、[むみま｜道具道](https://note.com/mumima) の「道具道」的関心から生まれた。


- トレンドを追うのではなく、**道具の生態系の変化を観測する**
- 特に「メディア生成系 / 知識管理系 / パイプライン接続系」の3ジャンルに関心がある

設計と実験は **Claude（Anthropic）との対話** を通じて行った。  
Claudeが実際にコードを書き、GitHubのライブデータで動作確認済み。  
その結果に自信を持って、このドキュメントをIDEへの実装依頼書として公開する。

## ドキュメント構成

| ファイル | 内容 |
|---------|------|
| `README.md` | このファイル。プロジェクト概要 |
| `SPEC.md` | 要求仕様。IDEへの実装依頼書 |
| `EXPERIMENTS.md` | 試したジャンルと、Claudeによる動作確認の記録 |
| `FEASIBILITY.md` | IDEで実装できるか？判断材料まとめ |
| `REPORT_TEMPLATE.md` | 出力レポートのテンプレート（実績は今後追記） |

---

## 対象ジャンル（v1）

1. **メディア生成系**（Remotion型）— コードで動画・音声・画像を生成するツール
2. **知識管理系**（Obsidian型）— ローカルファースト・Markdown・PKMツール
3. **パイプライン接続系**— 上記の前後に接続できる変換・自動化ツール
